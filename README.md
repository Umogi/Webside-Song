#Music Website

This project is a responsive and visually engaging website featuring various music genres, with embedded playlists and tracks for genres such as Phonk, Brazilian Phonk, and Pop. The website is designed for easy navigation with a sliding sidebar menu.

##Features

 - Responsive Design: Mobile and desktop-friendly.
 - Sidebar Navigation: Click to expand the sidebar menu for easy navigation.
 - Embedded Playlists: Direct links to playlists on Spotify.
 - Hover Effects: Interactive hover effects on images.

##Files

 - Main.html: HTML structure for the website.
 - styles.css: CSS styling for layout and design elements.
 - Java.js: JavaScript functionality for opening and closing the sidebar navigation.
